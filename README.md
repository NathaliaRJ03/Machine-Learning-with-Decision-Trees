🌳 Machine Learning with Decision Trees
This project explores the implementation of a Decision Tree model using Python and Scikit-Learn, covering the complete machine learning workflow: data preprocessing, model training, visualization, and performance evaluation.

📌 Project Overview
The objective of this project was to understand how Decision Trees work in practice and how they make predictions through a sequence of decision rules.

During the development, I explored key machine learning concepts such as:

Data preprocessing
Categorical variable encoding with LabelEncoder
Train-test split
Decision Tree training and tuning
Tree visualization using Graphviz
Model evaluation and interpretation
Overfitting prevention through tree constraints

🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-Learn
Graphviz

🌲 Decision Tree Analysis
One of the most valuable parts of this project was visualizing the generated Decision Tree and understanding its structure.

Key concepts explored:

Depth: the number of decision levels the model goes through before reaching a prediction.
Leaf Nodes: the final nodes where predictions are made.
Max Depth: controls how deep the tree can grow.
Max Leaf Nodes: limits the number of final prediction nodes.
Overfitting: understanding how overly complex trees can memorize training data instead of learning patterns.
